利用者台帳・Google Sheets 操作ガイド

台帳の構成・日常操作・国保連テンプレートの使い方
前提条件：GWS管理者設定ガイド（オプション①）が完了し、上村から台帳ファイルが共有されていること。

このガイドでできるようになること

  • 利用者台帳（5シート）の役割と使い方を理解できる
  • 新規利用者の登録・既存情報の更新ができる
  • 出退勤の月次集計を確認できる
  • 国保連テンプレート（実績記録票・給付費請求書）でデータを集計・出力できる
  • 個別支援計画の更新スケジュールを管理できる
1

利用者台帳マスターの5シート構成

–

台帳は1つのファイル（5枚のシートを持つ）

Google Sheetsでは1つのファイルの中に複数の「シート（タブ）」が持てます。利用者台帳は以下の5シートで構成されています。画面下のタブをクリックして切り替えます。

① 利用者マスター
② 出退勤記録
③ 記録DB
④ 国保連集計
⑤ 計画書管理

① 利用者マスター

利用者全員の基本情報を管理するシートです。

  • 氏名・生年月日・住所・連絡先
  • 障害の種別・等級・受給者証番号
  • 利用開始日・担当支援員・緊急連絡先
  • 利用状況（継続中 / 利用終了 / 休止中）

② 出退勤記録

アプリ①から自動で書き込まれる出退勤データシートです。

  • 日付・利用者名・出勤時刻・退勤時刻・利用時間
  • 月末に自動で合計利用時間を集計
  • 欠席・キャンセルの記録も含む

③ 記録DB

アプリ②（音声記録）やPhase 0（Mode A）で作成した支援記録が蓄積されるデータベースです。

  • 利用者ID・日付・記録種別・内容・担当者
  • 個別支援計画書の下書き作成にここのデータを使います

④ 国保連集計

毎月の国保連請求データを集計するシートです。②出退勤記録から自動でデータを集計します。

  • 月次の利用日数・利用時間・報酬単位の集計
  • 実績記録票テンプレートへのデータ出力元として使用

⑤ 計画書管理

個別支援計画書の作成・更新スケジュールを管理するシートです。

  • 前回の計画書作成日・次回更新予定日（6か月後）
  • 1か月前になるとGASから自動でメール通知が届きます
  • 計画書のDrive保存URLも記録
2

日常の操作手順

1

新規利用者を登録する（①利用者マスター）

アプリ③（受入れフォームアプリ）が稼働していれば自動で追加されますが、手動で追加する場合は以下の手順です：

  1. 台帳を開き、「①利用者マスター」タブをクリック
  2. 一番下の空白行をクリックしてカーソルを置く
  3. 各セルに情報を入力（氏名→生年月日→住所→…の順）
  4. 利用者IDは「U001」「U002」のように連番で付ける
  5. 入力後は上書き保存（Ctrl+S）— Sheetsは自動保存されますが念のため
利用者の氏名・住所・電話番号は個人情報です。印刷や外部への送付は行わないでください。
2

出退勤データを確認する（②出退勤記録）

アプリ①が稼働していれば自動記録されます。日々の確認手順：

  1. 「②出退勤記録」タブをクリック
  2. 今日の日付の行を探す（行は日付順に並んでいます）
  3. 全利用者の出退勤が記録されているか確認する
  4. 打刻漏れがある場合は上村に連絡するか、該当セルに手動入力
管理者は毎日15〜16時に確認する習慣をつけると、退勤漏れを当日中に確認できます。
3

月末の国保連データを集計する（④国保連集計）

毎月月末（または翌月1〜3日）に行う作業です：

  1. 「④国保連集計」タブをクリック
  2. 月末処理ボタン（上村が設置するGASボタン）をクリック → 自動で当月データが集計される
  3. 集計結果をざっと確認（利用日数・時間が実績と合っているか）
  4. 「国保連書類」フォルダの「実績記録票テンプレート.xlsx」を開く
  5. 集計データを実績記録票に転記する（または上村が作成したマクロで自動転記）
第2段階（4か月目〜）からはGASで国保連PDF自動変換まで対応します。第1段階はSheetsへの転記のみで進めます。
4

個別支援計画の更新期限を確認する（⑤計画書管理）

  1. 「⑤計画書管理」タブをクリック
  2. 「次回更新予定日」の列を確認
  3. 今月中・来月に更新期限が来る利用者をピックアップ
  4. Gemini（または記録DBのデータ）を使って個別支援計画の下書きを作成する（Phase 0 Mode Aの手順参照）
更新1か月前にGASから自動でメール通知が届くよう設定されています。メールが届いたら優先的に対応してください。
3

国保連テンプレートの使い方

1

実績記録票テンプレートを開く

Drive → 「国保連書類」フォルダ → 「実績記録票テンプレート.xlsx」を開きます。

テンプレートは毎月コピーして使います。元のテンプレートを書き換えないように注意してください。
  1. ファイルを右クリック → 「コピーを作成」
  2. ファイル名を「実績記録票_2026年05月」のように変更
  3. コピーしたファイルを開いて編集する
2

利用者ごとの実績を入力する

④国保連集計シートの数値を実績記録票に転記します：

  1. 利用者名・受給者証番号をマスターシートから確認して入力
  2. 利用日ごとにカレンダー形式で出勤に◯をつける（または自動マクロで入力）
  3. 合計利用日数・単位数が自動計算されることを確認
  4. 担当支援員のサインを確認欄に記入
実績記録票は国保連への請求根拠書類です。誤記があると返戻（差し戻し）になります。入力後は必ず管理者が確認してください。
3

給付費請求書を作成する

実績記録票をもとに給付費請求書を作成します：

  1. 「給付費請求書テンプレート.xlsx」をコピーして月別ファイルを作成
  2. 利用者ごとの単位数合計を入力
  3. 単位数 × 地域加算率 = 金額が自動計算される
  4. 印刷してサインを記入 → 国保連に提出（または電子請求システムへ入力）
レベル2（第2段階）では、GASでこの作業が半自動化されます。第1段階はテンプレートへの手入力で進めます。
4

よくある操作ミスと対処法

Q

セルを間違えて書き換えてしまった

Ctrl+Z（Undo）で直前の操作を取り消せます。保存後でも「ファイル → 変更履歴 → 変更履歴を表示」から過去の状態に戻せます。

Google Sheetsは変更履歴が自動で保存されています。完全に消えることはありません。
Q

数式が壊れた（エラーが出た）

#REF! や #VALUE! などのエラーが出たら上村に連絡してください。数式は上村が管理します。セルを空白にして保存しておいてください。

Q

他のスタッフが同時に編集しようとしている

Google Sheetsは複数人が同時に編集できます。誰かが編集中のセルは色がつきます。同じセルを同時に編集しないよう声かけして対応してください。

わからないことがあれば気軽にご連絡ください。
