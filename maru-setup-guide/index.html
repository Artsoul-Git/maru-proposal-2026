AI活用 導入ガイド 総合インデックス

有限会社アートソウル（上村桂右）が伴走サポートします ／ 2026年5月

このガイドは、合同会社まるのスタッフ全員がAIを活用した業務改善をスムーズに進められるよう作成しました。IT初心者の方でも安心して読めるよう、専門用語をなるべく使わず、一つひとつの手順を丁寧に説明しています。

ガイドはフェーズ別（Phase 0〜3）とオプション別（オプション①③）に分かれています。Phase 0からはじめて、順番に進めていきましょう。

Phase 別ガイド — アプリ導入手順
Phase 0

GWSセットアップ ＋ 即効録音体制

Week 1〜2（開発ゼロ・設定だけ）
Google Workspaceを導入し、Google Meetで面談を録音。Geminiに貼り付けるだけで記録が完成します。
  • Google Workspaceの初期設定
  • Google Meetで字幕・録音を使う
  • Geminiで面談記録を自動生成（手動コピペ）
  • Driveで記録を整理・共有
→ このガイドを読む
Phase 1

受入れフォーム ＋ 出退勤アプリ

Week 2〜4（アプリ③→①の順で開発）
新規受入れ手続きと出退勤記録をアプリ化。紙やExcelへの手入力が不要になります。
  • 受入れフォームアプリの使い方
  • 出退勤打刻アプリの使い方
  • Google Sheetsの台帳で一覧管理
→ このガイドを読む
Phase 2

音声入力 記録アプリ

Month 2（アプリ②）
スマートフォンのマイクに向かって話すだけで日々の支援記録を入力できます。
  • スマホで音声入力する手順
  • 記録アプリへの保存方法
  • あとから確認・修正する方法
→ このガイドを読む
Phase 3

AI記録 自動生成システム

Month 2〜3（アプリ④ GAS＋Gemini API）
面談のテキストが自動でAIに送られ、個別支援計画の記録が自動生成されます。
  • 自動生成の仕組みを理解する
  • 生成された記録の確認・編集方法
  • 個人情報マスキングの仕組み
  • エラー時の対処法
→ このガイドを読む

GWSアカウントの設定・権限管理・Driveフォルダ構成・Meetの文字起こし確認・Geminiテンプレート設置。

  • Driveフォルダの構成と確認方法
  • Meetの文字起こしが動くか確認
  • スタッフのアクセス権限管理
→ このガイドを読む

  • 5シートの役割と日常操作
  • 新規利用者の登録手順

  • 週次レポートの作り方（5分）
  • オンラインセッションの準備
  • 業務改善アイデアの見つけ方
  • KPI数値の記録・自走度評価
→ このガイドを読む

導入の全体的な流れ

1

Phase 0 — まず「すぐに効果が出る」体制を作る（Week 1〜2）

アプリ開発は後回し。Google MeetとGeminiだけで、面談記録が60分→15分になります。

2

Phase 1 — 入力の手間を自動化する（Week 2〜4）

新規受入れと出退勤をアプリで管理。日々の転記ミスや二重入力がなくなります。

3

Phase 2 — 音声で記録する（Month 2）

キーボード入力が苦手な方も、話すだけで記録できます。

4

Phase 3 — AIが記録を自動生成する（Month 2〜3）

面談後に何もしなくても、AIが記録の下書きを作ってくれます。確認・修正するだけです。

わからないことがあれば、いつでもご連絡ください。
