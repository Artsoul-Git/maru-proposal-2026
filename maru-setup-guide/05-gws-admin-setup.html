オプション① ／ Week 1〜2

GWS管理者設定 ＋ Driveフォルダ整備

上村が設定・まる様管理者が確認するガイドです
このガイドの使い方：各ステップに「上村が行う」「まる様管理者が行う」のタグを付けています。まる様側の担当者は まる様 のタグが付いた手順を実施してください。

オプション①でできるようになること

  • Google Workspace 3アカウントを業務用として正しく設定できる
  • Driveフォルダが整理され、スタッフ全員が迷わずアクセスできる
  • 利用者台帳（5シート）が使える状態になる
  • 国保連テンプレートでデータを集計・出力できる
  • Geminiで個別支援計画の下書きが作れるプロンプトテンプレートが手元にある
1

GWSアカウントの初期設定を確認する

1
上村が実施

GWSアカウントの発行・設定

上村が管理者コンソールで3アカウントを作成し、権限設定を行います。以下が完了したら各アカウント情報をまる様にお渡しします：

  • 3アカウントの作成（管理者1名 + AIマネージャー候補2名）
  • Google Workspace Business Standard プランの設定確認
  • ドメイン設定（既存メールアドレスと連携するか、新規ドメインを使うか）
  • Meet録音・文字起こし機能の有効化確認
2
まる様が実施

受け取ったアカウントにログインする

上村からお渡しするアカウント情報（メールアドレス・仮パスワード）でChromeからログインしてください。

  1. Chromeを開き、右上の人型アイコン → 「別のアカウントを追加」
  2. お渡ししたメールアドレスを入力
  3. 仮パスワードを入力 → ログイン
  4. 「パスワードを変更してください」と表示されたら、新しいパスワードを設定
パスワードは8文字以上・英数字+記号を含めてください。メモをとって安全な場所に保管してください。
3
まる様が実施

2段階認証を設定する（セキュリティ強化）

個人情報を扱うアカウントのため、2段階認証（ログイン時にスマホで確認する仕組み）を設定します。

  1. 右上のアカウントアイコン → 「Googleアカウントを管理」
  2. 左メニューの「セキュリティ」をクリック
  3. 「2段階認証プロセス」→「使ってみる」
  4. スマホの電話番号を登録し、SMSで届く確認コードを入力
設定後は、ログイン時にスマホにコードが届きます。不正アクセスを防ぐ重要な設定です。

フォルダへのアクセスを確認する

drive.google.com を開き、「共有アイテム」または「マイドライブ」に「まる_業務共有フォルダ」が表示されているか確認してください。

フォルダが見えれば共有設定は完了です。見えない場合は上村に連絡してください。
よく使うフォルダは右クリック → 「ショートカットをマイドライブに追加」で素早くアクセスできます。
3
まる様が実施

新しい利用者フォルダを追加する手順（覚えておく）

新規利用者を受け入れた際は、受入れフォームアプリ（アプリ③）が自動でフォルダを作成します。手動で作成する場合は以下の手順です：

  1. 「個人フォルダ」フォルダを開く
  2. 「新しいフォルダ」ボタンをクリック
  3. フォルダ名：「利用者氏名（仮名可）」で作成
  4. フォルダ内に「記録」「書類」「個別支援計画」の3サブフォルダを作成
アプリ③（受入れフォームアプリ）が稼働すれば、この作業はすべて自動になります。
3

Meet 文字起こし機能が使えることを確認する

1
上村 + まる様

テスト会議で文字起こしを動かす

上村との初回ミーティングで、実際にMeet文字起こしを体験します。以下を確認してください：

  1. meet.google.com を開いて「新しい会議」を作成
  2. 画面下部「その他のオプション（…）」→ 「文字起こし」→「文字起こしを開始」
  3. 数分話す（「本日はテストです。山田さんの面談記録を作ります」など）
  4. 会議を終了する
  5. 5〜10分後に Drive → 「Meetの録音と文字起こし」にファイルが作成されることを確認
Google Docsファイルが自動保存されれば設定は正常です！
「文字起こし」メニューが表示されない場合は管理者コンソールで機能が有効化されているか確認が必要です。上村に連絡してください。
4

Geminiプロンプトテンプレートを使えるようにする

1
上村が実施

Driveにプロンプトテンプレートを設置

上村が以下のテンプレートをDriveの「プロンプトテンプレート」フォルダに設置します：

  • 面談記録テンプレート（サービス提供記録・相談支援記録・議事録 各種）
  • 個別支援計画書下書きテンプレート
  • フェイスシートテンプレート
2
まる様が実施

Gemini でテンプレートを使う基本手順

  1. Chromeで gemini.google.com を開く（GWSアカウントでログイン済みの状態で）
  2. Driveの「プロンプトテンプレート」から使いたいテンプレートを開く
  3. テンプレートの文章をコピーし、Geminiの入力欄に貼り付ける
  4. 【記録データ】の部分に、Meetの文字起こしテキストを貼り付ける
  5. 送信ボタンを押す → 記録の下書きが生成される
  6. 生成されたテキストをコピーし、Google Docsに貼り付けて保存
詳しい使い方はPhase 0ガイドの「Section 4：Geminiで面談記録を生成する」を参照してください。
5

管理者として覚えておくこと

–
まる様管理者が実施

スタッフのアクセス権限を管理する

Driveの共有フォルダに新しいスタッフを追加・削除する場合の手順です：

  1. 対象フォルダを右クリック → 「共有」
  2. 追加する場合：メールアドレスを入力 → 権限を選択（「閲覧者」か「編集者」）→ 送信
  3. 削除する場合：現在の共有相手一覧から対象者を選択 → 「アクセスを削除」
退職したスタッフのアクセスは必ずその日のうちに削除してください。個人情報漏えい防止のための最重要作業です。
–
まる様管理者が実施

Googleアカウントのパスワードを忘れた場合

スタッフがパスワードを忘れた場合の対応手順：

  1. 管理者アカウントで admin.google.com を開く
  2. 「ユーザー」→ 対象スタッフを選択
  3. 「パスワードをリセット」→ 仮パスワードを設定して本人に伝える
  4. 本人が次のログイン時に新しいパスワードに変更する
管理者コンソールの操作が不明な場合は上村に連絡してください。
わからないことがあれば気軽にご連絡ください。
